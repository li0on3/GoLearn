# Day 2 练习说明

本目录包含切片深入学习的所有练习代码。

## 文件说明
- `slice_basics.go` - 切片基础操作练习
- `slice_traps.go` - 切片常见陷阱示例
- `dynamic_array.go` - 动态数组实现
- `slice_algorithms.go` - 切片算法练习
- 对应的 `*_test.go` - 测试文件

## 运行方式
```bash
# 运行特定文件
go run slice_basics.go

# 运行所有测试
go test -v

# 运行基准测试
go test -bench=.
```
